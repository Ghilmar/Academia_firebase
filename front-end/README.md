# Academia de Educación Profesional y Mentoría (React + Tailwind)

Proyecto base: aplicación web moderna, responsive, construida con React y Tailwind CSS.

Estructura propuesta (carpeta `src/`):
- components/
  - Header.js
  - MentorCard.js
  - AuthForm.js
- pages/
  - Home.js
- index.js
- index.css
- App.js

Archivos de configuración:
- package.json
- tailwind.config.js
- postcss.config.js
- public/index.html

Instalación rápida (ejemplo con npm):
1. npm install
2. npm start

Tailwind ya está preconfigurado en `tailwind.config.js` y `postcss.config.js`. Las imágenes de ejemplo usan Unsplash; en producción sustituir por tus recursos.
